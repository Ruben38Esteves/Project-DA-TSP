Edge * Vertex::findEdge ( Vertex *  dest) const

Find an edge connecting this vertex to the specified destination vertex.

This function searches for an edge in the adjacency list of the current vertex that connects it to the specified destination vertex. If an edge is found, it is returned; otherwise, nullptr is returned.

Time complexity: O(1)
